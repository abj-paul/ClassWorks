Checkbox with backend issue.

name-values/variables are single-valued. They are not arrays. They can only take a single value. When using checkbox, a user may select multiple option. In that case, if we use a single variable, it has to contain multiple values, like an array. But html does not support arrays. So in checkbox, we will have a separate variable/name for each checkbox.
